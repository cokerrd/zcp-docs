title: Marketplace
---

ZCP Marketplace images are pre-configured VM templates with applications installed and ready to use
on first boot. Deploy any image from the ZCP console and connect within minutes — no manual
installation required.

## Databases

| Image                              | Description                                                            |
| ---------------------------------- | ---------------------------------------------------------------------- |
| [MongoDB 8.0](./mongodb)           | Document-oriented NoSQL database for flexible, JSON-like data storage  |
| [MariaDB 11.4](./mariadb)          | Open-source relational database and drop-in MySQL replacement          |
| [MySQL 8.4](./mysql)               | The world's most widely deployed open-source relational database       |
| [PostgreSQL 17](./postgresql)      | Advanced open-source relational database with rich extensibility       |
| [Valkey 9.0](./valkey)             | High-performance in-memory data store, Redis-compatible                |
| [Elasticsearch 8](./elasticsearch) | Distributed search and analytics engine built on Apache Lucene         |
